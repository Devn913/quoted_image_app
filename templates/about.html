Hey Dev here, this is one of my small projects. What does it do? Well, it's just an open-source free to generate a quoted image (write a random quote on the image). Images source is Unsplash, Quotes source is Quotable API, and I have used Pillow python library to edit quote on images, for frontend I've used Tailblocks and combining all of them using Django and BOOM here's the app. Fun fact - click on the top right corner to see the source code behind this website.
